# Four In A Row
Four In A Row game written in C++

The purpose of this branch is data collection, and neural network training/integration.

## Prerequisites for compilation

CMake must be installed

The Eigen library for C++ must be installed

# Build instructions
## How to compile 
	$ git clone git@github.com:BrianO-04/Four-In-A-Row.git
	$ cd Four-In-A-Row	
	$ mkdir build
	$ cmake .S . -B build
	$ cmake --build build
## How to run
    $ cd build
	$ ./FourInARow
