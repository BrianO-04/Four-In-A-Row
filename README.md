# Four In A Row
Four In A Row game written in C++

## Prerequisites for compilation

CMake must be installed

# Build instructions
## How to compile 
	$ git clone git@github.com:BrianO-04/Four-In-A-Row.git
	$ cd Four-In-A-Row	
	$ mkdir build
	$ cmake .S . -B build
	$ cmake --build build
## How to run client
    $ cd build
	$ ./FourInARow
