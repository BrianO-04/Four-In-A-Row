# Four In A Row
Four In A Row game written in C++

Neural Network is currently on the 'neural' branch.

## Prerequisites for compilation

CMake and a C++ compiler must be installed

# Build instructions
## How to compile 
	$ git clone git@github.com:BrianO-04/Four-In-A-Row.git
	$ cd Four-In-A-Row	
	$ mkdir build
	$ cmake .S . -B build
	$ cmake --build build
## How to run
    $ cd build
	$ ./FourInARow
